Help

The questions and options are way more powerful than they look.

Instead of just taking text, it actually takes HTML code to allow you to have any kind of elements which a browser can display.
You can not just make image questions, you can also make videos and other cool stuff.
If you know how to write HTML go ahead and have a field day!
If you don't, but you still want to use images and links and simple HTML, you could use on of the many WYSIWYG editors, which are like Word.
You simply type in an use buttons to add complex elements, and then simply copy over the HTML in the correct locations.

Some online WYSIWYG editors that we like-

html-online html5-editor htmlg.com

If you need to show some media like images/videos, the easiest way to do it is to follow the following steps

  1. Upload your file to Google Drive
  2. Change the Share option to "Anyone With Link"
  3. Right click and Preview the file
  4. Click the ⋮ button in the top right
  5. Click Open in new tab
  6. Click the ⋮ button in the top right
  7. Click Embed item...
  8. Copy the displayed HTML code and paste it in the questions/options input
